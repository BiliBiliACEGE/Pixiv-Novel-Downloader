# Pixiv小说下载器

一个基于PyQt6的图形界面工具，用于从Pixiv下载小说内容。

## 功能特性

- 支持单本小说下载
- 支持批量下载多本小说
- 支持系列下载
- 可选择保存格式(TXT/HTML/Markdown)
- 下载历史记录功能
- 简洁美观的UI界面
- 多语言支持(简体中文/英文/日文)

## 使用说明

1. 安装依赖:
```
pip install -r requirements.txt
```

2. 运行程序:
```
python main.py
```

3. 使用界面:
- 在"单本下载"标签页输入小说ID或链接
- 在"批量下载"标签页可输入多个ID或链接(每行一个)
- 点击"设置"按钮可更改保存路径和文件格式
